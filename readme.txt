== Description ==

= sxss Mobile =
Detects if a visitor uses a mobile device and changes the actual theme to a special theme optimized for mobile devices (phones, tablets etc.). You can choose the layout, your menus and even customize the colors and you don't have to change the code of your standard Wordpress theme.

== Installation ==

- Upload the sxss Mobile plugin to your blog
- Activate the plugin
- Go to Appearance > sxss Mobile
- Install one of the recommended mobile themes
